# ЗЯ Трекер - Android приложение

Приложение для отслеживания заказов и километража курьеров.

## Функционал

✅ **Календарь** - главный экран с календарем для учета смен
✅ **Графики работы** - поддержка 5/2, 2/2, 3/3, 2/2/3 с настраиваемой датой начала отсчета
✅ **Подсветка рабочих дней** - рабочие дни подсвечиваются ярким цветом на календаре
✅ **Добавление смен** - добавление и удаление рабочих смен
✅ **Учет данных** - количество заказов и километраж за каждый день (1-350 км, 1-150 заказов)
✅ **Редактирование** - редактирование данных прямо в календаре
✅ **Переключатель рабочий/выходной** - любой выходной можно сделать рабочим днем
✅ **Сводка за месяц** - итоговое количество заказов и километров
✅ **Темная и светлая тема** - переключатель в настройках
✅ **Уведомления** - ежедневные напоминания в 20:00 о необходимости внести данные
✅ **Оффлайн режим** - полная работа без интернета
✅ **Профиль курьера** - имя и график работы с датой начала отсчета

## Технологии

- **Kotlin** - основной язык
- **Jetpack Compose** - современный UI
- **Room Database** - локальная база данных
- **DataStore** - хранение настроек
- **WorkManager** - планирование уведомлений
- **MVVM архитектура** - чистая архитектура
- **Coroutines & Flow** - асинхронная работа

### Требования
- Android Studio Iguana или новее
- JDK 17
- Android SDK 34
- Минимальная версия Android: 8.0 (API 26)

### Добавление смены
1. Нажмите на любую дату в календаре
2. Введите количество заказов (1-150) и километры (1-350)
3. При необходимости включите "Сделать выходной рабочим днем"
4. Нажмите "Сохранить"

### Редактирование смены
1. Нажмите на дату с существующей сменой (отмечена точкой)
2. Измените данные
3. Нажмите "Сохранить" или "Удалить"

### Настройки
- Изменение имени
- Переключение темы (Светлая / Темная / Системная)
- Включение/отключение уведомлений

## Разрешения

- `POST_NOTIFICATIONS` - для отображения уведомлений (Android 13+)
- `SCHEDULE_EXACT_ALARM` - для точного расписания уведомлений

## Архитектура

Приложение построено по принципу **Clean Architecture** с использованием **MVVM паттерна**:

- **Presentation Layer**: Jetpack Compose UI + ViewModel
- **Data Layer**: Room Database + DataStore
- **Domain Layer**: Репозитории с бизнес-логикой

## Лицензия

Проект создан для личного использования.
